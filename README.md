TaskList 

Sistema desenvolvido afim de submissão a teste de Desenvolvedor Java Web

Proposta: 
- Desenvolver uma tasklist utilizando Java/MySQL.

Requisitos:
O usuário deve poder:
* Adicionar novas tarefas;
* Marcar e desmarcar o status de concluído;
* Editar o conteúdo da task;
* Deletar uma task;
* Versionamento com Git;
Obs.: uma task deve conter ao menos: título e status, podendo conter adicionalmente descrição, datas de
criação, edição, remoção e conclusão.


Ferramentas Utilizadas no Projeto:
* Maven - para o gerenciamento de Dependencias do Projeto 
* Hibernate (JPA) 
* JSF 
* Primefaces
* BootStrap
* CSS3
* HTML5

Arquiterura Utilizada: 
* MVC 
* TDD

Testes: 
* JUnit
